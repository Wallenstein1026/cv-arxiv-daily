AI Literature Agent

This app runs a one-shot Codex agent that retrieves AI-related papers from arXiv, writes ranked paper outputs, and emits a downstream-ready agent_payload.json for other group skills.

  • Skill entry: python run_skill.py --config skill_config.yaml
  • Agent entry: python run_agent.py --config agent_config.yaml
  • Main outputs: outputs/manifest.json, outputs/agent_payload.json
